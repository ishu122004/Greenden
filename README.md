# 🌿 Greenden - Responsive Plant Website

Greenden is a responsive plant e-commerce website built using **HTML5** and **Tailwind CSS**. The project provides a modern and clean user interface with multiple pages, making it suitable for showcasing plants and gardening products.

## 🚀 Features

- Responsive Navigation Bar
- Attractive Hero Section
- About Us Section
- Best Seller Products
- Customer Reviews
- Newsletter Subscription
- Product Listing Page
- Contact Page
- Mobile-Friendly Design
- Clean and Modern UI

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- Responsive Web Design

## Pages

- Home
- Products
- Contact
